The essence of chronic disease：long-term management, not a one-time treatment.
👉 Core value for patients:
Being “continuously cared for”, rather than “handled temporarily”
Making professional healthcare a part of everyday life
Fewer unnecessary in-person follow-up visits; less hassle, less anxiety
Early warnings for abnormal indicators and timely feedback from medical staff

👉 Core value for healthcare professionals:
Upgrading from “treating illnesses” to “managing patient populations”, reduce healthcare cost;
Continuous, real-world patient data
Using data to drive prevention, instead of waiting for complications
Building a trust-based, long-term chronic care system

慢病的本质是长期管理，不是一次诊断。

👉 对患者核心价值：
被“持续照顾”，而不是被“临时处理”
把专业医疗变成日常生活的一部分
不必要的线下复诊；少折腾、少焦虑
异常指标提前预警、医护的及时反馈

👉 对医护核心价值：
从“看病”升级为“管理患者群体”；
连续、真实的患者数据；
用数据驱动预防，而不是等并发症;
构建“信任型”的长期慢病护理体系.
