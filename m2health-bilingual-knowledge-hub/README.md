## Deploy to GitHub Pages / 部署到 GitHub Pages

1. Create a GitHub repository, for example `m2health-knowledge-hub`.
2. Upload all files in this package to the repository root.
3. Go to **Settings → Pages**.
4. Choose **Deploy from a branch**.
5. Select `main` branch and `/root` folder.
6. Save and wait for GitHub Pages to build.

中文步骤：

1. 新建 GitHub 仓库，例如 `m2health-knowledge-hub`。
2. 将本包所有文件上传到仓库根目录。
3. 进入 **Settings → Pages**。
4. Source 选择 **Deploy from a branch**。
5. Branch 选择 `main`，Folder 选择 `/root`。
6. 保存后等待 GitHub Pages 自动部署。
